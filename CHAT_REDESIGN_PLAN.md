# 香蕉牛奶机 — 聊天 APP 重做方案

## 确定的交互流程

```
桌面（拟物化）
  ↓ 点「聊天」图标
智能体列表（像通讯录，显示头像/名称/对话数）
  ↓ 点智能体
直接进入最近联系的对话（或创建新对话）
  ↓
聊天界面
  ← 左滑（或点 ☰）打开左侧边栏（历史对话列表）
  → 右滑关闭侧边栏
```

## 聊天 APP 重构要点

### 1. 左侧边栏（当前是右侧）
- 对话列表从右侧面板改为左侧面板
- 右滑手势打开左侧边栏（当前是右滑打开右侧）
- 面板内容：该智能体的所有对话列表（搜索/新建/重命名/删除）

### 2. 智能体列表改为通讯录风格
- 显示头像、名称、最后联系时间、未读红点
- 点击直接进入最近对话（不先显示对话列表）
- 对话管理通过左侧边栏完成

### 3. 清理无用的占位功能
- 功能盒保留但只放有实际功能的项目
- 美化/设置改为项目书描述的正确面板
- 其他占位项先隐藏

### 4. 拟物化UI增强
- 聊天背景透明设计
- 消息气泡样式更像真手机
- 状态栏显示真实时间/电量
