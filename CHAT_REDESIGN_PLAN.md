# 香蕉牛奶机 — 聊天 APP 重做方案

## Grill-me 访谈确认的完整规格

### 打开流程
```
桌面（拟物化）
  ↓ 点「聊天」图标
智能体列表（通讯录风格，头像/名称/未读红点）
  ↓ 点智能体
直接进入最近联系的对话（或新对话）
  ↓
聊天界面（透明背景）
  ← 右滑打开左侧边栏（历史对话列表）
```

### 功能盒（左右滑动翻页，全部打开独立页面）

| 功能 | 类型 | 说明 |
|------|------|------|
| 语音通话 | ❌ 暂缓 | |
| 设置 | 页面 | 折叠思考链开关、tool call列表 |
| 美化 | 页面 | 9项设置（背景/头像/气泡/时间/Token/头像框等，上传支持裁剪） |
| MCP配置 | 页面 | 启用/禁用开关（配置在设置APP） |
| 网络搜索 | 页面 | 启用/禁用开关（配置在设置APP） |
| 记忆 | 页面 | 独立管理页（总结/新增/修改/删除，注入系统提示词） |
| 主动消息 | 页面 | 事件监听推送，APP内弹窗+手机通知，推最近窗口 |
| 思考强度 | 页面 | 滑动条控制 reasoning_effort |
| 上下文拼装 | 页面 | 只读预览LLM完整输入，标注来源 |

### 搜索
输入关键词 → 匹配消息列表（高亮）→ 点击跳转

### 智能体设定
头像裁剪界面、名称、LLM模型、OCR模型、TTS、temperature、topP、系统提示词、世界书挂载、删除确认

### 消息
- 微信风格状态 ✓ / ✓✓ / ⚠️
- 思考链折叠（用户自选默认状态）
- 虚拟滚动 + 无限滚动
- 所有提示词可预览编辑
